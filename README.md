# Portal de Noticias - TP Final Taller de Internet

## Descripción

Portal de noticias desarrollado como Trabajo Práctico Final para la materia Taller de Internet.

El proyecto fue realizado utilizando únicamente HTML, CSS y JavaScript Vanilla, sin frameworks ni librerías externas.

## Funcionalidades

* Visualización de noticias en la página principal.
* Modo visitante sin necesidad de autenticación.
* Inicio de sesión mediante API DummyJSON.
* Persistencia de sesión mientras el navegador permanece abierto.
* Gestión de noticias (alta, modificación y eliminación).
* Persistencia de noticias mediante Local Storage.
* Modo oscuro con persistencia de preferencias.
* Visualización de la cotización del dólar mediante API Bluelytics.
* Publicación mediante GitHub Pages.

## Tecnologías Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)
* Local Storage
* Session Storage
* API DummyJSON
* API Bluelytics

## Cómo Ejecutar el Proyecto

1. Clonar el repositorio.
2. Abrir el proyecto en Visual Studio Code.
3. Ejecutar el archivo `index.html` utilizando Live Server o un navegador web.
4. Acceder al portal como visitante o iniciar sesión como administrador.
